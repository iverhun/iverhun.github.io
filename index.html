Tech Skills

Java

Python

🐍 Python is my best friend when it comes to scripting and pet projects for home usage, such as TV Photo Frame or IoT backend apps. Pandas 🐼 Numpy, and Matplotlib 📈 are great for data processing and analysis. And Turtle 🐢 is fun when we learn programming with my kids.

AWS Cloud Computing

Amplify, ALB, Cognito, CloudFront, CloudWatch, GuardDuty, ECS, KMS, Lambda, RDS… Phew… 🥵 Let me stop here, even though this is not the end of the list of the services I have used to build scalable, secure applications on top of AWS.

Docker

📦 Whether deploying in ECS or Kubernetes, testing with TestContainers, starting multiple services with Docker Compose, or keeping my hard drive clean by using yet another DB in a container, Docker is about simplicity, safety, and order.

Databases

Designing DB schemas, troubleshooting and optimizing queries, data migrations, and schema evolution - this is what I do with confidence 🧑🏽‍💻 My favorite RDBMSs are MySql and PostgreSQL, but I feel excited about exploring others SQL and NoSQL DBs such as MongoDBB or AWS DynamoDB.

Linux

When “talking” to a computer, I’m a CLI guy 🖥️, preferring text commands over UI controls. Linux offers the best CLI experience, powerful scripting capabilities, and a robust security model. Indeed, I’m not a Linux hacker, but my knowledge of Linux allows me to conduct training and mentor others 🎓

Event Driven Apps

I have been using JMS (ActveMQ), AMQP (RabbitMQ 🐰) and Kafka for enterprise apps, and MQTT for home-made IoT projects.

Git and CI/CD

I’m one of those who know exactly how Rebase in Git works, and I’m happily sharing this knowledge with others. I can also configure a build pipeline, whether it is GitHub, Bitbucket, or GitLab. The same applies to AWS CodeCommit, Azure Repos, or the old good Jenkins.

Agile

The Agile development process is like a game. Not only do I know the rules, but I also love to make them serve the team so that the team is productive and motivated. Scrum or KanBan depends on a few factors, but the focus is always on achieving the result.
